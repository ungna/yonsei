git remote add origin 주소
=> 내가 github의 repository를 연결할거다 이름은 origin이다

git pull origin main(master) branch이름이 main인지 master인지 잘보고 홈페이지껄로 적어야됨
=> 내가 연결할 주소에서 main(master)라는 이름의 branch를 가지고옴
